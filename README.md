# 🌸 Happy Birthday Shika ✨

A cinematic, interactive birthday website with smooth 3D animations and premium design.

## 🚀 Quick Deploy to Vercel

### Option 1: Deploy via Vercel Dashboard (Easiest)

1. **Go to [Vercel](https://vercel.com)**
2. **Sign up/Login** (use GitHub, GitLab, or Bitbucket)
3. **Click "Add New" → "Project"**
4. **Import your repository** or upload this folder
5. **Vercel will auto-detect Vite** - just click "Deploy"!
6. **Done!** Your site will be live in ~2 minutes

### Option 2: Deploy via Vercel CLI

```bash
# Install Vercel CLI globally
npm i -g vercel

# Navigate to project folder
cd shika-birthday

# Deploy
vercel

# Follow the prompts:
# - Set up and deploy? Y
# - Which scope? (your account)
# - Link to existing project? N
# - Project name? shika-birthday
# - Directory? ./
# - Override settings? N

# Production deployment
vercel --prod
```

## 🛠️ Local Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Open http://localhost:5173
```

## 📦 Build for Production

```bash
npm run build
```

The `dist` folder will contain your production-ready files.

## 🌐 Alternative Hosting

### Cloudflare Pages
1. Go to [Cloudflare Pages](https://pages.cloudflare.com)
2. Connect your Git repository
3. Build command: `npm run build`
4. Output directory: `dist`
5. Deploy!

### Netlify
1. Go to [Netlify](https://netlify.com)
2. Drag and drop the `dist` folder
3. Done!
